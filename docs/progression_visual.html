⑧ Empire Indices — What Feeds Them

SPI
Strategic Power Index
  • active_ship.combat_rating + fleet_pool/1,000
  • Sum of "defense" production (all colonies)
  • character_military_skill × 2
  • Military research progress / 100
  • Faction bonus: Allied Industry/Tech → +4/+12/+25
REI
Resource Extraction Index
  • minerals×8 + refined_ore×6 + visited_systems×2
  • ether×10 + AEF_stat/5
  • active_ships×12 + visited_systems×3
  • KIN/8 + SYN/10 + COH/12 (extraction aptitude)
  • AEF/6 + visited_systems (prospecting)
  • profession_level/10
KII
Knowledge & Innovation Index
  • research_output × 2 (all colony production)
  • ether_yield (all colony ether)
  • research buildings × 5 (count)
  • Researcher / Scholar profession bonus
ECI
Economic Capability Index
  • credits_per_turn × 0.5 (production + pop tax)
  • trade_volume (goods bought/sold via market)
  • food_surplus (production − consumption)
  • Colony economic/political system modifiers

⑨ Reverse Lookup — "How do I get X?"

↑ RP / turn (research output)
  1. INT stat → character RP bonus
  2. Scientist class → +25% research_speed
  3. Buildings (free): Academy of Sciences (+4)
  4. Buildings (gated): Synthetic Mind Core +10, Observatory +6, Neural Institute +5, Psych-Net Hub +4, Xenobiology +4, Academy +4, Quantum Relay +3, Research Node +3
  5. Planet: Crystal World (crystal terrain × research)
  6. Economic: Memory Economy +25%, Consciousness-Labor +20%
  7. Political: Consciousness Swarm +30%, Oracle-Mediated +20%
  8. Social: Memory-Pooled +20%, Distributed Selfhood +10%
  9. Faction: Allied Veritas Covenant → +50% speed
↑ Credits / turn
  1. Buildings: Interstellar Bank +20, Spaceport Hub +12, Trade Nexus +8, Luxury Habitat +5, Fabricator +6
  2. Terrain: Coastal tiles → credit ×1.2 modifier
  3. Population: More colonists → 75 cr per 10k/turn
  4. Economic: Probability Economy +20%, Energy-State +10%
  5. Political: Distributed Sovereignty +5% all_production
  6. Faction: Allied Stellar Nexus → −30% trade prices
  7. Class: Merchant Captain trade_discount +10%
↑ Fleet Points / SPI
  1. Buildings: Orbital Drydock +18, Shipyard +6×ore, Command Citadel +5, Munitions Factory +4, Barracks +2
  2. Fleet chain: Ore Processor → refined_ore → Shipyard ×(1+ore×0.05), cap +100%
  3. Economic: Consciousness-Labor +10% fleet_points
  4. Social: Resonance-Based Cohesion +10% fleet_points
  5. Faction: Allied with Industry/Tech faction → SPI +25
  6. Class: Military Commander +25% combat_effectiveness
↑ Ether output
  1. Buildings: Crystal Resonance Array +6 (crystal), Void Anchor +5 (void), Geothermal Tap +4 (geo/volcanic), Neural Institute +1, Psych-Net Hub +2, Tidal Plant +2 (coastal)
  2. Terrain: Void ×2.0, Crystal ×1.5, Geothermal ×1.3
  3. Planet type: Crystal World or Lava World
  4. Stat: High AEF → REI energy sub-score
↑ Ship combat power
  1. Crew: Weapons Officer +10 weapon_output, Pilot +12 maneuver, Shield Technician +10 shield, Chief Engineer +15 engine_eff
  2. Crew leveling: L10 = 1.5× all bonuses
  3. Crew morale: Medical Officer +15 morale → all bonuses ×1.25
  4. Station upgrades: Weapon/shield/engine components
  5. Class: Military Commander +25% combat_effectiveness
  6. Faction: Allied unlocks faction-locked hull/engine components
↑ Fuel efficiency / range
  1. Crew: Engineer +8 engine_efficiency, Chief Engineer +15, Pilot +8 engine_output
  2. Components: High engine_efficiency base value on hull/engine
  3. Class: Explorer +15% fuel_efficiency
  4. Avoid: Dangerous regions (+50% fuel), heavy cargo
↑ Faction reputation
  1. Gifts: 500 cr per point, max +20/action
  2. Colony systems: Match systems to faction preferences (up to +15/turn)
  3. Economic: Ritualized Exchange +10% to all rep gains
  4. Social: Narrative-Bound +15% rep gains
  5. Political: Consensus Field +10% rep gains
  6. Stat: High INF → Social Engineering → diplomatic effectiveness
  7. Class: Diplomat +30% diplomatic_relations
↑ Population growth
  1. Food buildings: Biofarm Complex, Jungle Harvester, Bio-Synthesis Lab, Agri-Industrial Complex, Atmospheric Processor
  2. Pop buildings: Population Hub +1.5%/each, Luxury Habitat +0.8%/each
  3. Economic: Ritualized Exchange +15%, Memory Economy +10%
  4. Social: Symbiotic Networks +20%, Narrative-Bound +10%, Memory-Pooled +5%
  5. Cap: Max 8% growth per turn
Unlock: Ether buildings
  1. Complete Advanced Etheric Weaving → Ether Conduit, Etheric Purifier, Crystal Resonance Array
  2. Complete Null Space Exploration → Void Anchor
  3. Complete Plasma Energy Dynamics → Geothermal Tap
